# 공통 오류 코드

모든 서울 버스 API에서 공통으로 사용하는 오류 코드입니다.

| 코드 | 설명                                 |
| ---- | ------------------------------------ |
| `0`  | 정상                                 |
| `1`  | 시스템 오류                          |
| `2`  | 잘못된 쿼리 요청 (파라미터 확인)     |
| `3`  | 정류소를 찾을 수 없음                |
| `4`  | 노선을 찾을 수 없음                  |
| `5`  | 잘못된 위치 요청 (위/경도 좌표 확인) |
| `6`  | 실시간 정보 읽기 실패 (재시도)       |
| `7`  | 경로 검색 결과 없음                  |
| `8`  | 운행 종료                            |
